# 🎓 Smart Campus Management System

A desktop-based application developed using **Java, MySQL, and JDBC** to automate and manage campus operations like student records, attendance, and feedback.

---

## 🚀 Features

- Student Record Management  
- Attendance Tracking System  
- Course & Faculty Management  
- Feedback Management  
- Real-time Database Operations using JDBC  

---

## 🛠 Tech Stack

- **Java (Core + JavaFX)** – Application development & UI  
- **MySQL** – Relational Database  
- **JDBC** – Database Connectivity  
- **Maven** – Project Management  

---

## 📂 Project Structure

```
src/main/java/
│
├── App.java
├── StudentDAO.java
├── AttendanceDAO.java
├── FeedbackDAO.java
├── LoginUI.java
├── MainUI.java
├── ViewDataUI.java
```

- DAO classes handle database operations  
- UI classes manage user interaction  
- Clear separation between logic and data layer  

---

## 🧠 Concepts Used

- DAO (Data Access Object) Design Pattern  
- Normalized Relational Database Schema  
- Foreign Key Relationships  
- JDBC Connectivity  
- Parent → Child Data Insertion Handling  

---

## ⚙️ How It Works

1. User interacts with the system through UI  
2. Java processes the request  
3. JDBC connects to MySQL database  
4. Data is stored/retrieved in real-time  
5. Results are displayed back in the UI
